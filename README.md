# Aplikasi Penjualan

Aplikasi penjualan berbasis web yang dibuat sebagai project kelas 11 semester 2.

## 🛠️ Teknologi
- HTML
- CSS
- PHP

## ✨ Fitur
- Login & autentikasi pengguna
- Input data penjualan
- Manajemen produk
- Laporan & rekap penjualan

## 📁 Cara Menjalankan
1. Clone repo ini
```bash
   git clone https://github.com/esselrn/aplikasi-penjualan.git
```
2. Pindahkan folder ke direktori server lokal (contoh: `htdocs` untuk XAMPP atau `www` untuk Laragon)
3. Import database dari folder `/database` (jika tersedia)
4. Jalankan **XAMPP** atau **Laragon**, aktifkan Apache & MySQL
5. Buka browser dan akses `http://localhost/aplikasi-penjualan`

## 📝 Catatan
Project ini dibuat untuk keperluan tugas sekolah dan tidak di-deploy secara publik.
